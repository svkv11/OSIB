## Overview

A Python-based Body Mass Index (BMI) Calculator that takes a user's weight and height as input and computes their BMI, then classifies their health category according to standard WHO guidelines.

---

## What is BMI?

**Body Mass Index (BMI)** is a numerical value derived from a person's weight and height. It is commonly used as a screening tool to categorize underweight, normal weight, overweight, and obesity in adults.

**Formula:**

```
BMI = weight (kg) / height (m)²
```

---

## BMI Categories (WHO Standard)

| BMI Range         | Category          |
|-------------------|-------------------|
| Below 18.5        | Underweight       |
| 18.5 – 24.9       | Normal weight     |
| 25.0 – 29.9       | Overweight        |
| 30.0 and above    | Obese             |
